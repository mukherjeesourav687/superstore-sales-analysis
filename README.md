# Superstore Sales Data Analysis

## Objective
Explore the Superstore dataset to understand sales patterns across different product categories.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed
- Dataset exploration using df.info() and df.describe()
- Category-wise sales analysis
- Data visualisation using bar charts

## Key Insight
Technology generates the highest sales among all categories.
